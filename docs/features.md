# Features Overview

## Core Product Features

### 1) Dashboard
- Risk-focused overview cards
- Parameter summary and health distribution charts
- Detected issue highlights and insight chips
- Local snapshot continuity for recent analysis

### 2) Report Analysis
- Upload medical report (PDF/image)
- OCR extraction pipeline
- Structured parameter generation and validation
- AI-generated summary with risk level and reasons

### 3) Chat Assistant
- General medical-information chat (`/chat` endpoint)
- Report-specific chat with context (`/report/chat` endpoint)
- Safety disclaimer embedded in responses

### 4) Knowledge Hub
- Topic-based educational articles
- Category filtering and article detail view
- "Explain in simple terms" handoff into chat flow

### 5) Hospital Finder
- Interactive map using Leaflet
- Overlay list and recommended hospitals
- Modal-based details and quick map focusing

### 6) Report History
- Review previously processed analyses
- Compare extracted insights and outcomes over time

## Key Capabilities
- OCR + parsing + AI in a single workflow
- Staged APIs for modular processing
- Risk explainability through concrete reason strings
- Responsive UI with desktop and mobile support
- Demo-friendly architecture with fallback resiliency
